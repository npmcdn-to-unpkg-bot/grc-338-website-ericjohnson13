Professional Interests

In the past several months, I have started to become increasingly interested in learning and teaching Ruby on Rails. Ruby on Rails is a facinating web framework that allows for rapid web application development. I've also learned that Ruby is indeed an incredible programming language that, as the developer has said, is designed for programmer productivity and fun.

Case in point, below are two block of code that do the exact same thing. The first is written in JavaScript, the second written in Ruby. I leave it to you to determine which is easier to use and understand.

JavaScript

            
              function rand(a, b) { return Math.floor(Math.random() * (b - a + 1) + a); }
              rand(5, 9); // one of [5, 6, 7, 8, 9]

              a = [b, b = a][0] // do not try at home :-)

Ruby

            
              prng = Random.new()
              prng.rand(5..9) # one of [5, 6, 7, 8, 9]

              a, b = b, a # switch a and b
